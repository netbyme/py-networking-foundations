# py-networking-foundations

Python labs I built to learn network automation from scratch.
Each lab uses real networking concepts like IP parsing, validation and device inventory.
Part of my path toward CCNA and Cisco DevNet certification.

## Labs

- Lab 01 - Device Inventory Basics
- Lab 02 - Device Inventory using Dictionaries
- Lab 03 - Dictionary Methods
- Lab 04 - Parse IP Addresses
- Lab 05 - Validate IP Format
- Lab 06 - Classify IP Ranges
- Lab 07 - Reusable Network Functions
- Lab 08 - Match IPs to Hostnames
- Lab 09 - Read and Write Config Files
- Lab 10 - Filter Active Devices
- Lab 11 - Transform Device Data
- Lab 12 - Parse Network Log Files
- Lab 13 - Handle Connection Errors
- Lab 14 - Read CSV Device List
- Lab 15 - Generate Network Inventory Report

## Skills built

- Python fundamentals — variables, lists, loops, dictionaries
- IP parsing, validation and classification logic
- Writing reusable functions for network tasks
- Automating device inventory with zip() and counters
- Reading and writing network config files
- Filtering and transforming data with map, filter, lambda
- Log file parsing and error extraction
- Error handling with try/except
- CSV file processing
- Generating automated network inventory reports

## Tools

- Python 3.12
- Ubuntu WSL
- VS Code
